# MailForge — Email Marketing Landing Page Template

A premium, fully responsive landing page template for email marketing platforms, SaaS products, and marketing tools.

## 🔥 Features

- **Dark theme** with modern glassmorphism design
- **Fully responsive** — looks great on desktop, tablet, and mobile
- **13+ sections** including hero, features, automation builder, templates, analytics, pricing, testimonials
- **Interactive elements** — workflow builder, template gallery, pricing toggle, modals, particle background
- **Animated counters**, scroll animations, and hover effects
- **Zero dependencies** — pure HTML, CSS, and vanilla JavaScript
- **Google Fonts** (Inter) for beautiful typography

## 📦 What's Included

- `index.html` — Main page with separate CSS/JS files
- `styles.css` — Complete styling (1000+ lines)
- `app.js` — All interactivity and animations
- `mailforge-complete.html` — Single self-contained file (everything bundled inline)

## 🚀 Quick Start

1. Download the files
2. Open `index.html` in your browser
3. Customize colors, text, and images to match your brand

## 🎨 Easy to Customize

- All colors defined as CSS variables in `:root`
- Clean, commented code structure
- Modular sections — easy to add, remove, or reorder
